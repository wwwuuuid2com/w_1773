<h3>源码简介：</h3>
<p>酷瓜云课堂，依托腾讯云基础服务架构，采用C扩展框架Phalcon开发，GPL-2.0开源协议，致力开源网课系统，开源网校系统，开源在线教育系统，全功能可免费商用。<p>
<p>系统功能
实现了点播、直播、专栏、会员、积分、秒杀、微聊等。<p>

系统配置低（1核 1G 1M 跑多个容器），切莫压测
